Install the following required libraries using the following commands
On windows:
1) pip3 install SpeechRecognition pydub
2) pip3 install pyaudio
3) pip3 install pyttsx3
4) pip3 install gTTS
5) pip3 install playsound


On ubuntu (20.04 - pip3 and python3) systems:
1) pip3 install SpeechRecognition pydub
2) sudo apt-get install portaudio19-dev python3-pyaudio
3) pip3 install PyAudio
4) pip3 install pyttsx3
5) pip3 install gTTS
6) pip3 install playsound
